## Feature: Custom index methods and operator classes

### What

The `hasIndex()` API now supports custom index methods (`gin`, `gist`, `hash`, `brin`, `spgist`) and operator classes (e.g., `gin_trgm_ops`), enabling trigram search indexes and other specialized PostgreSQL index types.

### Why

Substring/fuzzy text search (e.g., `ILIKE '%pattern%'`, `similarity()`) requires GIN or GiST indexes with trigram operator classes. Previously, `hasIndex()` only supported default B-tree indexes, forcing users to manage these indexes outside the ORM via raw SQL migrations.

### Usage

```typescript
// GIN trigram index for substring search
entity.hasIndex('idx_users_email_trgm', e => [e.email])
  .using('gin')
  .withOperatorClass('gin_trgm_ops');

// GiST index
entity.hasIndex('idx_users_name_gist', e => [e.name])
  .using('gist')
  .withOperatorClass('gist_trgm_ops');

// BRIN index (no operator class needed)
entity.hasIndex('idx_events_created', e => [e.createdAt])
  .using('brin');
```

### Details

- `IndexBuilder.using(method)` — type-safe union: `'btree' | 'gin' | 'gist' | 'hash' | 'brin' | 'spgist'`
- `IndexBuilder.withOperatorClass(opClass)` — validated against `^[a-zA-Z_][a-zA-Z0-9_]*$` to prevent SQL injection
- `IndexMethod` type exported from the package for consumer use
- `executeCreateIndex` refactored to accept an index definition object instead of 7 positional parameters
- Both `ensureCreated()` and migration diff detection propagate the new fields

- `src/entity/entity-base.ts` — added `IndexMethod` type, extended `IndexMetadata` with `using` and `operatorClass`
- `src/entity/entity-builder.ts` — added `using()` and `withOperatorClass()` to `IndexBuilder`
- `src/schema/table-builder.ts` — added `IndexMethod` type, extended `IndexDefinition` with `using` and `operatorClass`
- `src/migration/db-schema-manager.ts` — refactored `executeCreateIndex` to object param, added runtime validation, updated SQL generation with `USING` clause and operator class suffix
- `src/index.ts` — exported `IndexMethod` type
- `tests/schema/gin-index.test.ts` — 6 test cases covering GIN, GiST, BRIN index creation, trigram search, multiple indexes, and operator class validation
